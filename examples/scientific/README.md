# Scientific Examples

These are the benchmark-heavy, Rust-vs-C++ reproducibility demos:

- `examples/option1_quick_proof/`
- `examples/option2_balanced_science/`
- `examples/option3_large_scale_headline/`
- `examples/option4_sparse_stress/`
- `examples/option5_full_package/`

Global aggregate figure:
- `examples/comparison.png`
- `examples/comparison_summary.json`

Per-option outputs include:
- Dataset timing/metric curves (`*.png`)
- Baseline confusion matrices for classification datasets (`*_confusion.png`)
- Baseline residual diagnostics for regression datasets (`*_residuals.png`)

Run any option via its `run.sh`.
